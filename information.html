Indian Cattle Breed Encyclopedia

Explore detailed profiles of all cattle breeds used in our deep learning model — lifespan, weight, location, milk nutrients, and body features.

🎓 About the Project

This system is a Final Year B.E. project at MIT Academy of Engineering, Alandi, Pune. It uses a Convolutional Neural Network (CNN) trained on the Indian Cattle Image Dataset by Atharva Darpude (Kaggle) to identify Indian cattle breeds from images, videos, or live camera input.

96.4% Accuracy CNN Model Transfer Learning OpenCV

📦 Dataset Details

  • Source: Kaggle — atharvadarpude
  • Dataset: Indian Cattle Image Dataset
  • 10 major Indian cattle breeds
  • Thousands of annotated images
  • Multiple angles & lighting conditions
  • Govt. recognized indigenous breeds
  • Used for: Classification & Feature Extraction

⚙️ Model Pipeline

  • Image preprocessing via OpenCV
  • Data augmentation to reduce overfitting
  • CNN architecture (ResNet / MobileNet base)
  • 50 training epochs, batch size 32
  • Softmax output layer for breed probabilities
  • Python · TensorFlow / Keras
  • Frontend: HTML, CSS, JavaScript
